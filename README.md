# 🚀 Getting Started

Learn more: [nx.dev](https://nx.dev)

## Visualize Project Graph

```bash
npx nx graph
```

# ⚙️ Setup

1. **Install Dependencies**

```bash
npm install
```

# 📂 Explore the Workspace

- Show project graph:
  ```bash
  npx nx graph
  ```
- List all projects:
  ```bash
  npx nx show projects
  ```
- Show details of a specific project:
  ```bash
  npx nx show project angular-app
  npx nx show project react-app
  ```

# ▶️ Run Applications

## Angular App

Runs on: [http://localhost:4200](http://localhost:4200)

```bash
npx nx serve angular-app
```

## React App (Vite)

Runs on: [http://localhost:4201](http://localhost:4201)

```bash
npx nx serve react-app
```

_You can run both apps in parallel using separate terminals._

# 🏗️ Build Applications

- Build Angular app:
  ```bash
  npx nx build angular-app
  ```
- Build React app:

  ```bash
  npnx nx build react-app

  ```
